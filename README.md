AI Sales Voice Engine

Production-grade autonomous voice agent system.

Features

Latency: Sub-800ms response time using Retell + Cartesia.

CRM Integration: Bi-directional HubSpot sync for contact context and call logging.

Dialogue Control: Intent-based branching for objection handling.

Deployment

Install dependencies: pip install -r requirements.txt.

Set up environment variables in .env.

Run with python main.py.
